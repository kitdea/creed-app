# Creed Memorization App

An interactive fill-in-the-blank tool to help memorize the **Apostles' Creed** and **Nicene Creed**.

## Features

- 📖 **Read mode** — full text for study
- 🌱 **Easy** — ~25% of phrases blanked
- ✦ **Normal** — ~50% of phrases blanked
- 🔥 **Hard** — all phrases blanked
- ↺ Randomize blanks each round
- ✓ Check answers with score feedback
- 👁 Reveal individual wrong answers
- 🌙 Dark / ☀ Light mode
- Fully responsive (mobile to desktop)

## Getting Started

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
```

## Tech Stack

- React 18
- CSS-in-JS (inline styles)
- Google Fonts: EB Garamond + Cinzel
